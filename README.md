# Yay-Packer

![Screenshot_20230831_220432](https://github.com/krafairus/Yay-Packer/assets/64279814/179e647a-5737-4637-803f-d43fc4fac79f)

Es una herramienta versátil que puede utilizarse para instalar, desinstalar, actualizar y administrar paquetes a través de yay mediante una interfas grafica usando python y qt.

Nota: actualmente solo permite, instalar, desinstalar paquetes y actualizar todos los paquetes del sistema usando yay, tambien debido a que optiene los resultados de la busquedas de los paquetes con yay, al buscar cosas de forma no tan directa como solo buscar la eltra h, sobre carga la aplicacion y provoca que se congele, esto es debido a que debe mostrar muchos resultados, la solucion es que especifique mejor la busqueda.
